# dropboxlike-server
`dropboxlike-server` is a web framework based on Tornado and sqlite database. Maybe it should be called transporterlike,



## Getting Started

Make sure you have successfully installed Python 2.7
```
pip install -r pip-req.txt
python start.py
```
